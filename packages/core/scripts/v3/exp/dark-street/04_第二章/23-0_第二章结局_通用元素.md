# 第二章：结局

## 通用收尾元素（所有分支都必须包含）

1. **罗英的诉求暂告段落**：无论主角如何回应她的拉拢，罗英都会表示"时间还有，但不多了"。她会给出一个期限——第三章的时间节点。届时骷髅会回利用那批消失的军火，对黄昏会发起'战争'。

2. **骷髅会的威胁明确化**：激进派系的存在被确认。他们不会在第二章直接动手，但会在收尾处留下明确的威胁信号（如暗街某处被人搜查过的痕迹）。

3. **卡琳娜的关系更新**：根据第二章的所有互动，卡琳娜对主角的关系会发生变化。她会用一个动作或一句话，让主角感知到这种变化。

4. **卡尔的台词**：在第二章的结尾，卡尔必须有一句关于"选择"或"代价"的台词，为第三章的主题做铺垫。
