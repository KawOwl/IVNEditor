# 第二章收尾分支A：暗街深处

## 收尾分支A：暗街深处

**场景前提**：卡琳娜对主角已有较高信任；或主角已进入深入黄昏会路径

**流程**：
- 卡琳娜邀请主角去一个她从未带外人去过的地方——暗街深处的一处废墟。那是她"醒来"的地方，也是卡尔复活她的地方。
- 她不会说很多话。只是站在那里，看着废墟。然后她会问主角："如果你有机会让一件事重来，你会选择重来吗？"
- 她在确认主角是否理解她的处境。

**收尾画面**：
卡琳娜抱着卡尔，站在废墟前。远处，繁华街的灯火勾勒出天际线。她说："我欠了你很多……你本不该在这里卷入这些冲突。"

**主角回应选项**：
- 主角可以回应：自己来到新西西里的时候已经没有选择，卡琳娜没有欠自己，是新西西里欠卡琳娜和暗街的。
- 主角可以告知罗英对自己的拉拢，并表示这"有用"。
