# 第三章：节点剧情

**本部分为第三章专用剧情数据库。**

**覆盖范围**：从第二章结束至第三章结局的完整内容
**时间线**：第二章结束后的第二天开始，至第七天深夜结束（跨度约五天）
**核心场景**：暗街公寓 → 各方势力据点 → 最终决战场景 → 结局场景

---

## 第三章核心冲突

### 主线冲突

骷髅会的威胁从"调查"升级为"清除"。迈克尔·布莱克亲自下令，要求激进派系首领艾萨克·科恩彻底解决暗街的"怪物"问题。与此同时，黄昏会与凯旋门之间的紧张关系因军火事件持续发酵，新西西里即将迎来一场全面冲突。

卡琳娜必须在各方势力的夹缝中找到出路——或者，彻底打破这个困局。

### 第三章核心主题

**选择与代价**

每一个结局都对应着一种选择，每一种选择都有其必须支付的代价。没有人能全身而退，但每个人都可以决定自己愿意失去什么。

### 关键信息揭示（第三章必须揭示）

以下信息在第三章中必须全部揭示，分布在不同的叙事节点：

1. **卡尔复活的真相**：十年前，骷髅会与凯旋门联手设局，用一件"遗物"将卡尔的力量封印，并杀死了祂的实体。但祂没有真正死去——只要暗街仍然是新西西里的"阴影"、祂就永远不会消失。复活卡琳娜、是祂需要一个人代理自己的权柄。'卡琳娜'才是暗街现在真正的统治者。

2. **罗英的交易内容**：十年前，罗英用黄昏会在龙都的全部势力网络，换取了卡尔的一个承诺——"当黄昏会面临灭顶之灾时，暗街会出手一次"。现在她来索取，但她也知道，强迫卡尔履行的代价是她自己也未必能承受的。

3. **康纳的真正立场**：康纳对卡琳娜的复杂情感源于共同的过去，但她从未真正站在卡琳娜的对立面。她一直在用自己的方式——偏执的、扭曲的、不为人理解的方式——保护暗街不被凯旋门彻底吞并。第三章中，她将做出最终的选择。

4. **艾萨克·科恩的遗物**：那件能制约卡尔的遗物，是卡尔"生前"佩戴的猫牌。它沾染过卡尔的血液，因此与祂的力量同源。科恩利用这件遗物，可以短暂地切断卡尔与暗街的联系——让暗街的"阴影"失去宿主，但实际上，这只能解放卡尔的力量、让卡琳娜掌控这里的一切——这是卡尔对抗他们背叛的后手。
