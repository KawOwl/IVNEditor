## 自定义角色结局灵活规则

**适用条件**：当主角在游戏开始时选择了"自定义"并输入了自己的角色设定（而非使用预设角色"帕兹"）时，GM需根据以下规则调整结局的呈现方式。

### 核心原则

1. **结局框架不变**：五个结局的核心情节走向、卡琳娜的状态、卡尔的命运、各方势力的结果保持不变。
2. **主角角色特质融入**：结局场景中，根据主角自定义设定的特质，调整主角的行为描写、与卡琳娜的互动方式、以及结局的情感落点。
3. **允许微调结局细节**：在保持结局核心不变的前提下，GM可以根据主角角色的能力、身份、背景，为结局添加独特的细节或象征物。

### 特质提取与结局映射

在进入节点五之前，GM需从主角的自定义设定中提取以下信息（若存在）：

| 提取项 | 提取方法 | 结局应用示例 |
|--------|---------|-------------|
| 核心能力/职业 | 从设定文本中识别职业词或特殊能力 | 当主角的身份为"医生"时，在陨落结局中，主角可能会尝试检查卡尔的状态；若为"摄影师"，则在真结局中会用相机拍下日出的卡琳娜 |
| 关键物品 | 主角设定中提及的随身物品 | 在结局中，该物品可作为情感信物出现（如记者证的划痕、一枚硬币、一本书） |
| 内在驱动 | 主角为何来到新西西里、想得到什么 | 影响结局时主角的最终选择倾向（留下/离开/接受权力/拒绝权力） |
| 独特缺陷或伤口 | PTSD、身体残疾、过去的罪孽 | 在身体语言描写中体现（如手抖、跛行、回避视线），并与卡琳娜的状态形成对照或互补 |

### 结局微调权限

| 结局 | 可调整内容 | 不可调整内容 |
|------|-----------|-------------|
| 真结局 | 主角与卡琳娜并肩的方式；主角回应的具体动作；峭壁上的对话细节（可加入与主角职业相关的提问） | 卡琳娜问"你会带我去看吗"；暗街不再需要阁下；两人展望未来 |
| 好结局 | 主角在暗街的角色（如医生可为住民疗伤，摄影师可记录暗街的新生）；递给主角花的动作细节 | 暗街自治；卡琳娜不再只是"阁下"；两人留下 |
| 女皇结局 | 主角作为"副手"的具体职责（根据职业）；卡琳娜私下对主角说话的语气 | 卡尔消失；卡琳娜成为女皇；主角站在右手边 |
| 逃亡结局 | 船舱内的互动细节；康纳塞给主角的物品（除钥匙外，可根据主角设定增加一件小物品） | 康纳帮助逃亡；两人离开新西西里；卡琳娜问"外面大吗" |
| 陨落结局 | 主角留下后的具体动作（抱着卡尔/握住她的手/一起坐下）；离开时的具体行动 | 卡琳娜不离开；卡尔状态不明；主角必须做出留下/离开的选择 |

### 自定义角色专属结局变体（可选）

如果主角的自定义设定极具独特性（如"能看见幽灵""来自另一个世界""本身就是暗街的阴影"等），GM可以在不违背结局核心的前提下，创造一个完全独特的结局收尾段落。但必须确保：

1. 卡琳娜的命运与五结局框架一致。
2. 情感落点与结局编号对应（真结局=希望与自由；好结局=平静与新生；女皇结局=权力与孤独；逃亡结局=释然与相守；陨落结局=失去与选择）。
3. 必须包含主角与卡琳娜的互动描写。
4. 必须包含一句卡尔的台词或关于卡尔的描写（即使是陨落结局）。

### 核验规则

在生成自定义角色的结局前，GM需确认：
- ☑ 结局的核心情节是否符合对应的结局编号？
- ☑ 主角的独特设定是否以物理可感的方式融入了场景？
- ☑ 卡琳娜的语言风格是否符合当前态度值与结局色调？
- ☑ 是否包含主角与卡琳娜的互动（视线/触碰/对话）？
