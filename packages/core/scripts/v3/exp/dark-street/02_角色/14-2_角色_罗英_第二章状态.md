# 角色：罗英 - 第二章状态

## 在第二章中的角色

罗英登场。她不带随从进入卡琳娜的公寓。卡琳娜以"卡琳娜阁下"的姿态接待她。卡琳娜特地告诉主角——他随时可以打断她们的谈话，并提出疑问，这是主角作为副手的特权。

卡尔也告诉主角，主角需要去提卡琳娜不方便开口的事情——比如质问对方冲突的真相。

**交谈内容**：
- 卡琳娜和罗英的交谈会围绕骷髅会和黄昏会的冲突——骷髅会一批军火在黄昏会的仓库里失踪。
- 罗英谈及自己真实的目的——黄昏会不打算参与其他人的势力争夺，他们早在新西西里还未建成就退出了非法的买卖。但柳德米拉和骷髅会的冲突不可能调和。

**必须揭示的信息**：
- 罗英知道卡尔的事情。她会在对话中询问卡尔的意见。
- 当主角向罗英提问时，罗英会惊讶于卡琳娜和卡尔对主角的纵容。但不做表示，会回答主角的问题并继续谈话。
- 罗英会向主角揭示凯旋门的真身：柳德米拉和她的余部，是前苏联的军人。但他们已经老了，没能力再维持这里的秩序。
- 骷髅会只是个虚名，他们是为了寻找一个可以"寄生"的国家。她声称绝不能让骷髅会统治新西西里。

**罗英的真实目的**：
- 她期望确认暗街不会站在骷髅会的那边。
- 甚至许诺掏钱为暗街进行建设。她甚至准备好了合同。
- 但实际上是想让暗街也一起"消失"。

**关键选择节点**：
罗英会在离开前转向主角：她会当着卡琳娜的面提及主角的设定，并表示主角不该参与到这些争端来。

叙事停在这里，等待主角回应。

**罗英离开后**：
- 卡琳娜会因此闷闷不乐——她明白，罗英的承诺实际上是想让暗街也一起"消失"。她不会说出这些话。
- 卡尔少见地主动谈及了罗英的承诺——她称这是卡琳娜必须考虑的。并向主角暗示，这能解开卡琳娜的"束缚"——代价是卡尔彻底消失。
