# 角色：罗英 - 第三章状态

## 第三章状态更新

罗英在第三章中代表"旁观者"的立场。她不是敌人，也不是朋友。她是那个在风暴边缘观望、计算、等待的人。

但她有自己的原则。当卡琳娜拒绝她的"退路"提议，选择正面面对时，罗英第一次露出了某种接近于敬意的表情。

**她对主角的态度**：
- 她认可主角的判断力——如果主角在与她会面时表现出对卡琳娜的理解。
- 她不会和主角做交易，但她会记住主角说的话。

**她会做的**：
- 保持黄昏会的中立。
- 在结局之后的世界里，记住这份人情——并在此后某个意想不到的时刻偿还。
- 在陨落结局中，当主角决定离开时，她的人会在港口等。这是她唯一能做的。
