# 角色：艾萨克·科恩 - 第三章决战中

## 第三章决战中的科恩

**决战核心场景：港口区地下岩洞**

科恩选择这里作为陷阱的地点，因为军火库深处有一个天然的岩洞——那里的"阴影"最浓，最适合遗物发挥作用。

**科恩的台词**：
- "你以为这就能阻止我？"（康纳站队后援迟迟不到时）
- "我研究这个东西研究了十年。我知道怎么杀死它。"
- "你果然来了。把它带来。"
- "现在是我的。十年前，你的血沾在上面的时候，它就不再属于你了。它属于杀死你的人。"

**遗物激活**：
科恩咬破自己的拇指，将血涂在猫牌上。猫牌上的锈迹开始剥落，露出下面暗金色的金属本色。红光变得刺眼。

**科恩的失败**：
- 卡尔的身体几乎完全透明了。
- 但就在科恩以为胜利在握的那一刻，岩洞深处的阴影开始动了。
- 墙壁上的影子脱离了岩壁的束缚，像活物一样蔓延开来。它们爬上军火箱，爬上科恩的腿，爬上他握着猫牌的手。
- 科恩的笑容僵住了。
- "这是什么……这是什么？！"
- 猫牌不只是能消除卡尔、它也是一个信标。
- 科恩的手臂已经被阴影完全覆盖。他的手指一根根松开。猫牌落在地上，红光熄灭。
- 科恩跪在地上。他的嘴唇在发抖，但发不出声音。

**如果遗物被激活（失败路径）**：
卡尔的力量被切断。但暗街的阴影失控——卡尔控制的阴影，失去了宿主的、原始的、饥饿的阴影、它们涌入了卡琳娜体内。
岩洞开始崩塌、现实本身开始被阴影吞噬。科恩第一个被吞没。
