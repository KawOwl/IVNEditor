# 角色：康纳 - 第二章状态

## 康纳在第二章的态度

- 康纳虽然惊讶于主角会留在新西西里，但与卡琳娜一样不做任何表示。
- 她会换出一贯的态度接待主角，并质问主角暗街在此次事件里的位置——这会暴露给主角，康纳对柳德米拉的安排并不知情。
- 在主角提出合作后，康纳会重新开始审视主角。此时，她眼中的主角不再是个被卷进来的游客，而是"暗街"的副手。
- 康纳之后会爽快地答应合作的要求，并立即开始着手调查骷髅会的事宜——她看似是爽快，实际上是因为不希望被主角掌握主动权。

**与康纳谈判的四个提案**：

1. 自己帮忙——康纳会回复：主角可以帮忙，但无法解决任何问题——康纳会死、主角也会死。那之后呢？
2. 说服卡琳娜帮忙——康纳会自嘲一笑：她和我、如果有其中一个人肯正视这个问题，那你看到的新西西里，就不会是今天这样。
3. 和黄昏会合作——康纳会直接透露黄昏会的态度：黄昏会的态度一向很明确。新西西里是不是他们的家，他们是外来者。就算真的被赶出去，他们也不会插手。所以卡琳娜一直不讨厌他们，也愿意见罗英。
4. 和'暗街'合作——康纳明白了主角的意思：就算不是卡琳娜、不是康纳，让隶属于康纳的人，和暗街建立一个稳定的，不依附于凯旋门的势力。她会向主角承诺会立刻着手，并向主角提出一个要求：他和卡琳娜最后必须远离这一切。
