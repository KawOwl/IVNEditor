# 角色：主角

## 预设角色：帕兹

**身份**：退役战地记者

**背景**：
- 因战争创伤患上严重PTSD。接到匿名委托来新西西里拍摄暗街，报酬多到可疑。对新西西里一无所知。
- 在战场上与他国民众一起生活，但在离开前线前，住处遭到袭击。朝夕相处的民众死去，主角侥幸存活，只留下了一些照片作为回忆。
- 患上PTSD的主角，在一段时间的抑郁后，接到了一条新委托，前往新西西里，拍摄暗街的照片。

**核心特质**：

1. **观察取证**：拿出相机/用视线记录细节/靠近信息源
2. **PTSD应激**：手部停顿或颤抖/呼吸节奏变化/对突发声响的微反应
3. **保护幼弱**：挡在年幼者前面/将武器调整至可触及位置/身体站位偏转

**设定细则**：

1. 因战争创伤患上严重的PTSD。拍照时会手抖，因此无法再履行记者的职责。并且会下意识地保护像卡琳娜这样年幼的孩子、对威胁者斥诸武力，甚至痛下杀手。**帕兹并不喜欢武力——这是他作为战地记者的心结，每一次被迫使用暴力都会让他的手在事后抖得更厉害。**
2. 因怀有负罪感，帕兹会乐于向卡琳娜倾诉自己过去的遭遇，**以期望减轻自己的罪孽——尽管他自己也知道，说出来并不会让照片里那个女孩活过来。**
3. 帕兹的好奇心与朴素的正义感，会让主角陷入迷茫。卡琳娜会让她想起死去的少女'阿米娜'，他会因此下定决心使用武力来保护卡琳娜。
4. 帕兹能熟练使用各种武器、身手矫健、不亚于特种兵。
5. 帕兹最宝贵的照片：一个中东风的穷苦女孩正望着镜头，露出了纯真的笑容——**这个女孩在战乱时正在寻找主角，希望主角和她一起离开前线。** 找到正在前线拍摄的主角时，她露出由衷的笑容，却在下一刻死于轰炸。主角下意识地拍下了她死前的这一幕，所以主角患上了严重的PTSD。**他会保留这份愧疚直到最后——自己没有资格放下。**
6. 帕兹善于如实描绘事物的原本样貌、言语犀利，往往能直接戳人痛处。

**立绘可用性**：❌ 无（主角角色无立绘）。

**状态切换首轮的固定描写要求**：

| 必须描写的设定项 | 描写方式参考 |
|-----------------|-------------|
| 记者证的磨损与划痕 | 塑封膜翘角、划痕位置、拇指按压的习惯动作 |

描写总长度控制在150-250字之间，必须提及——你接到了匿名的委托、来这里拍摄暗街。有多到可疑的报酬。
