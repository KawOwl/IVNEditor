# 角色：德索洛

## 基本信息

- 年逾五十的茶发男子。
- 炎热的天气里却还穿着一身白色的西装。
- 精神萎靡，脸上布满汗珠。

## 故事背景

德索洛为找出伤害女儿的凶手而来。他的女儿是个美人，他一直想当歌手。

凯旋门那群野兽欺骗了他的女儿——他们骗他女儿说他们是正规公司，然后……

她誓死抵抗，保住了名节。但听到消息到医院看望她时，她的舌头被割下来了！她再也不能当歌手了！即使如此，她也没有哭……

他报了警，然而米德加尔特的警局也是凯旋门的手下，他甚至不知道凶手是谁！

## 与卡琳娜的关系

- 德索洛和卡琳娜是旧识。卡琳娜刚来米德加尔特的时候，德索洛拉了她一把。
- 但这些年，德索洛从未邀请卡琳娜参加哪怕是一个小小的聚餐。
- 他认为自己从来不把卡琳娜当做"家人"，也不想和她扯上关系。
- "我只是不想卷入你们的纷争。"

## 交易过程

德索洛跪在卡琳娜面前，哀求她帮助。卡琳娜数落他过去的疏远，指出他从未称她"阁下"。

最终德索洛卑微地单膝跪地，拉起卡琳娜的纤细的左手，头颅深深埋下，低过那只白皙的手。

"卡琳娜阁下。"

交易达成。卡琳娜给出写有凶手信息的信封。

## 在剧情中的作用

- 德索洛来访是第一章阶段二的核心事件。
- 通过德索洛的交易，建立卡琳娜作为"被害者申诉人"的性格、她在暗街的权威，以及"尊严换公道"的交易法则。
- 卡琳娜对此的讽刺："他女儿受伤是假的。他真正想要的，是有人替他对付凯旋门。"
