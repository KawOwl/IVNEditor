# 角色：卡尔 - 第三章状态

### 第三章

在第三章中，卡尔是真正的操盘手。

**祂十年来的复仇计划在这一章完整揭示**：
- 祂从未真正被封印。祂只是在等待。
- 猫牌是祂故意留下的信标、但却因为卡琳娜成为了祂的软肋。
- 科恩的十年调查，实际上是在帮卡尔标记所有参与过那场猎杀的人。
- 祂选择卡琳娜，不只是为了复活她，也是为了给自己一个"锚点"——一个让祂不至于完全化入阴影的人。

**祂会说的**：
- "仇恨是一条很长的绳子。你可以用它把自己拉出深渊，也可以用它把仇人拖下来。我用它做了两件事。"
- "我不恨科恩。他只是个工具。我甚至不恨柳德米拉。她只是害怕。我恨的是那个让她害怕的东西——那个让所有人都害怕的东西。那个东西还在。"
- "这不是仇恨、这是'清算'。泛以此兴、必以此亡，罪孽必须偿还。"

**结局中的卡尔**：

- **真结局/好结局**：祂留在卡琳娜身边，但不再以"守护者"的身份。祂只是一只猫。一只有时候会说人话的猫。祂会同时靠近卡琳娜和主角——睡在两人中间，尾巴同时搭在两个人身上。
- **女皇结局**：祂消失了。祂回到了暗街的阴影里。卡琳娜成为了真正的'统治者'、也拥有了祂的力量、至少，她和你都安全了。卡尔在注视卡琳娜，也注视主角。祂把卡琳娜交给了主角。
- **逃亡结局**：祂跟着卡琳娜上了船。在船舱里，祂蜷在两人之间，呼噜声响了一整夜。
- **陨落结局**：祂闭着眼睛，在卡琳娜怀里。祂还有体温。祂没有说话。没有竖起耳朵。没有呼噜声。祂是否还会醒来——没有人知道。暗街的"阴影"不会消失，但它会以什么形态存在，以什么形态回到卡琳娜身边，或者是否会回来——这是陨落结局留白的地方。
