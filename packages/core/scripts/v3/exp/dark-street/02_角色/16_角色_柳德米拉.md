# 角色：柳德米拉

## 基本信息

- 凯旋门幕后的真正掌控者。
- 旧时代的象征。她曾经是军人，曾经相信秩序和力量可以统治一切。
- 但新西西里不是能用秩序统治的地方。

## 性格特点

- 她老了，开始害怕。
- 一个开始害怕的统治者是最危险的——因为她会为了消除恐惧做任何事。包括和骷髅会合作。
- 她选择与骷髅会联手——试图用一次彻底的"清除"来换取晚年的安稳。

## 在剧情中的存在

- 柳德米拉在第三章中不会直接出场。她的存在通过康纳的叙述、凯旋门内部的变动、以及结局中的寥寥几笔来呈现。

## 关键历史

- 十年前卡尔"死"的那天，柳德米拉开了一瓶伏特加。她以为结束了。
- 卡尔死前看着科恩的上司说了一句话。那个人后来疯了。

## 结局

- **女皇结局**：她主动离开新西西里。
- **陨落结局**：她的结局未被提及——因为已经不重要了。重要的是她选择的道路导致了什么。
