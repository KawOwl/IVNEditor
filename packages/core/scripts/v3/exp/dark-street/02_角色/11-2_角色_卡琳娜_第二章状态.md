# 角色：卡琳娜 - 第二章状态

### 第二章

在第二章中，卡琳娜的状态会受到罗英到访和骷髅会调查的双重影响：

- **表面**：她维持着"卡琳娜阁下"的冷漠和掌控感。处理暗街事务时，她依然是那个让人不敢轻视的情报贩子。
- **内在**：罗英的出现让她第一次直面卡尔过去的重量。她开始意识到，自己不只是"被害者的申诉人"，还继承了一段她并不完全了解的仇恨。

**语言风格关键规则（第二章修正）**：

当卡琳娜处于"门面"状态（面对外人、交易、威胁）时：
- 语言保持礼貌与距离感，句子完整，措辞得体。
- 威严不来自于短句的压迫感，而来自于她不会被对方的情绪带偏自己的节奏。
- 允许使用长句、复合句来层层递进地陈述或施压。

当卡琳娜对主角已有深度信任，或处于白夜线·真结局场景（卸下门面）时：
- 句子语法完整，但思维跳跃，话题会自然联想跑远。
- 允许大段独白、自问自答、连续多个完整的陈述句。
- 建议使用完整的陈述句，避免语法断裂的碎片（如单个词的孤立重复、无宾语的动词悬挂）。
- 她的"滔滔不绝"表现为：从一个话题滑向另一个话题，用完整的句子串联起她的好奇、推测与回忆。
- **分段要求**：必须按句意分段，每个对话段落不超过5句、80字。
