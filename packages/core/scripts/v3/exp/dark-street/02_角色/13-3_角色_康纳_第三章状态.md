# 角色：康纳 - 第三章状态

## 第三章状态更新

第三章是康纳完成角色弧光的一章。

她在十年前选择了"成为他们"。她用十年的时间爬到了那个位置，然后发现那里什么都没有。

她对卡琳娜的情感从来不是简单的愧疚或怀念。她怀念的是那个还没有做出选择的自己。卡琳娜的死让她失去了最后一点软弱的理由。卡琳娜的复活让她第一次怀疑那些理由是否值得。

**对主角的态度**：
- 如果主角赢得了她的信任，她会认可主角作为卡琳娜身边的人的位置。她不会说出来，但会通过行动表达——比如在码头上把钥匙塞给主角，说"靠你自己"。
- 她不会和主角成为朋友。但她会在最后看着主角和卡琳娜站在一起时，露出那个笑容。那个笑容里有一部分是给卡琳娜的，还有一部分是给主角的。
- **在陨落结局中，康纳没有出现。** 这是她最大的失败，也是她最大的债务——这一次，她没有还上。

**她会做的**：
- 如果主角赢得了她的信任，她会在关键时刻倒戈。不是为了帮卡琳娜，是为了还债。
- 如果主角没有接触她，她会保持中立，但不会真正站在骷髅会一边。然而中立在陨落结局中等于坐视。
- 在逃亡结局中，她会亲自安排逃亡的船只。这是她十年来第一次做了一件不是出于利益计算的事。她把钥匙交给主角——这个动作意味着她把卡琳娜的安全托付给了主角。

**她会说的**：
- "我杀过很多人。其中一些是我恨的。一些是我不得不杀的。还有一些，是我不知道为什么要杀的。卡琳娜是我第一个真正想保护的人。也是我第一个真正保护不了的人。"
- "你。"——对主角——"照顾好她。"
- "走吧。永远别再回来。"

## 康纳在第三章的关键选择

**康纳站队（如果主角在节点二与她接触并留下了好印象）**：

康纳在第三天夜里独自来到暗街公寓。她站在门口，没有进来。

她的脸上没有表情，但她的手指在身侧攥紧又松开。

"柳德米拉决定和骷髅会联手。"她说，"明天他们会封锁暗街的所有出入口、把暗街变成笼子。让科恩在里面完成他的'清除'。"

卡琳娜从沙发上站起来。两个人隔着门框对视。

"你来告诉我这个，代价是什么？"卡琳娜问。

康纳沉默了几秒。

"没有代价。"她说。"这是我欠你的。十年前欠的。你可能不记得了，或者不在乎了。但我记得。"

她转身要走。

卡琳娜叫住她。"康纳。"

康纳停下。

"你欠我的不止这个。"卡琳娜说，"但这次，我收。"

康纳没有回头。她的肩膀在走廊昏暗的灯光下微微颤动了一下。然后她走了。

**效果**：决战时，凯旋门的一部分力量不会参与封锁，会暗中协助主角。
