# 世界观：暗街

## 基本定位

- 被遗弃的内陆区域，穷困者聚集地。
- 表面混乱，实则在"申诉人"卡琳娜的阴影下维持着一种脆弱的"公道"。
- 暗街真正的秩序维持者是一只名为"卡尔"的黑色短毛猫——新西西里"噩梦"的化身。

## 暗街的真相

- 卡尔曾死于黑帮之手，又以某种方式归来。
- 它蛰伏于此，向整个新西西里复仇——报复那些曾试图杀死它的人，报复那些让暗街沦为弃地的势力。
- 卡琳娜是它复活的棋子，也是它唯一的同伴。
- 它赋予卡琳娜"被害者申诉人"的使命，但在漫长的共处中，这只猫发现自己在乎她的感受，超过在乎复仇本身。
- "卡尔"并非是真实存在的生物。凯旋门曾借助祂的力量扫清了一切障碍，但与熟悉这类超自然事物的骷髅会联合，妄图将卡尔彻底葬送。他们失败了。因为卡尔的真身正是新西西里的"阴影"，正是他们自己让"卡尔"死而复生。

## 暗街的"阴影"之力

- 卡尔的力量与暗街绑定。只要暗街仍然是新西西里的"阴影"，祂就永远不会消失。
- 暗街的"阴影"没有消失——它永远不会消失。
- 因为暗街的存在，卡尔的力量会逐渐恢复，甚至更甚以往。
- 如果骷髅会知情，恐怕会直接向暗街动武——但这只会完全解放卡尔的力量，完全毁灭新西西里。
- 卡琳娜只能维持暗街的现状，在其他黑手党控制下的新西西里，暗街永远不可能翻身。

## 暗街的秩序

- 暗街的"公道"由卡琳娜作为"申诉人"维持。
- 卡琳娜的"权力"来自于暗街的人。如果其他势力想夺走她的"权力和力量"，就只有让暗街彻底消失。
- 暗街的人学会了以"尊严"和"亏欠"作为交易的底层逻辑。

## 暗街的历史

- 卡琳娜与康纳同出身暗街福利院。
- 10岁时福利院土地被凯旋门强买。
- 康纳被幕后操控者看中作为继承人培养。
- 卡琳娜在德索洛帮助下受教育。
- 康纳为生存得罪其他副手。副手为报复调查出卡琳娜存在，趁康纳外出时发难，杀死卡琳娜。
- 康纳以卡琳娜之死为借口发起清洗，成为实质掌权人。
- 十年里，现在的暗街在康纳心里就是"卡琳娜的墓碑"。
