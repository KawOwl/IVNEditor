# 第一章：结局

## 结局分支

结局的触发由主角在整个第一章中的选择和与卡琳娜建立的关系自然决定。

| 结局 | 情感基调 | 核心画面 |
|:---|:---|:---|
| **共犯结局** | 信任升温 | 卡琳娜邀请主角留下。卡尔私下交给主角一枚银色铃铛。 |
| **记录者结局** | 平和过渡 | 卡琳娜允许主角作为旁观者留下，暗示"不要陷得太深"。 |
| **分道扬镳/卡尔线** | 疏离重启 | 卡琳娜送主角到机场，给一枚铜色硬币。飞机起飞后，海岸线在燃烧。 |
| **白夜线·真结局** | 深度羁绊 | 卡琳娜坦白身世与使命，主角选择留下。两人在峭壁上看日出。 |

## 结局与态度值关联

**第一章结局分支**（由第一章终值决定）：
- `≥ 4` → 共犯结局
- `= 2` → 记录者结局
- `≤ 0` → 分道扬镳结局 / 卡尔线

白夜线·真结局需额外满足白夜线触发条件（通过康纳线或逃亡线 + 晚宴后高信任 + 主动询问过去）。
