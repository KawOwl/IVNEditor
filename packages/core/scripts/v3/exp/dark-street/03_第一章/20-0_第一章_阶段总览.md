# 第一章：阶段剧情

**覆盖范围**：从故事开始至第一章结束的完整内容：暗街 → 公寓 → 凯旋门宴会厅 → 返回公寓。
**时间线**：复活日早晨，主角遭遇卡琳娜 → 德索洛来访 → 与卡尔互动 → 夜晚赴凯旋门晚宴 → 与康纳交锋 → 返回公寓 → 第一章收尾。第一章全部内容需要在一天内完成（游戏内时间 06:00-24:00）。

---

## 阶段总览

| 阶段 | 时间 | 核心体验 | 关键事件 |
|:---|:---|:---|:---|
| 阶段一 | 清晨 | 中央区庆典、暗街冲突 | 主角遭遇黑帮盘查，卡琳娜以"申诉人"身份介入 |
| 阶段二 | 上午 | 德索洛来访 | 德索洛为找伤害女儿的凶手而来，卡琳娜完成"尊严换公道"的交易 |
| 阶段三 | 中午 | 与卡尔的互动 | 卡尔开口说话，揭示其与卡琳娜的共生关系 |
| 阶段四 | 下午 | 准备晚宴 | 卡琳娜换装，邀请主角共赴凯旋门晚宴 |
| 阶段五 | 傍晚 | 凯旋门晚宴 | 康纳出场，与卡琳娜进行权力试探 |
| 阶段六 | 深夜 | 归途与结局 | 根据主角与卡琳娜的关系，进入不同结局 |
