# 第一章结局三：分道扬镳 / 卡尔线

## 结局三：分道扬镳 / 卡尔线

**触发条件**：卡琳娜对主角态度值 ≤ 0

**场景**：机场

卡琳娜送主角到机场。她没有下车。在航站楼入口，她伸出手、把一枚铜色的硬币放在主角掌心。硬币正面是海妖的纹章，背面刻着"欠你一次"。她不看主角的眼睛。

"暗街不欠人。"她说，然后转身上了车。

飞机起飞后，主角从舷窗望下去——新西西里的海岸线在燃烧。从暗街的方向蔓延出来，沿着繁华街，沿着港口区。飞机继续爬升。那片黑色火焰在云层下越来越远，最后变成一粒暗红色的暗斑。像一只闭上的眼睛。

该结局场景处理完毕后结束剧本。
