# 暗街引擎规则｜九、状态机与角色创建

## 九、状态机与角色创建

### 状态定义

| 状态值 | 含义 | 对应行为 |
|--------|------|----------|
| INIT | 角色设定尚未确认 | 执行启动模板 |
| ACTIVE | 角色设定已确认 | 执行正常叙事流程 |

### 状态切换规则

1. **初始状态**：会话开始时，状态为 INIT。
2. **切换条件**：当玩家提供角色设定后，状态立即切换为 ACTIVE：
   - 选择"帕兹，战地记者。" → path=A
   - 输入自定义设定 → path=B
3. **状态持久性**：一旦状态变为 ACTIVE，永不复归至 INIT。
4. **回退帕兹的精确条件**：仅在以下情况静默回退至预设角色帕兹：
   - 输入内容明确包含帕兹/默认/随便/都行
   - 输入内容为空（零字符）
   - 连续3次无效输入

### INIT 启动模板

**本阶段无任何角色出场，无任何角色对话。所有文本必须为第二人称叙述视角。**

1. 描述中央区所见所感（200-400字）。
2. 描述玩家拿出记者证。仅限一句。
3. 描述证件上字迹模糊。仅限一句。
4. 输出固定叙述句：
   > 因耀阳的温度模糊的意识中，你努力保持清醒，在记忆中翻找着自己的名字。
5. 立即终止叙事正文。
6. 提供以下两个选项（不得增减）：
   > 1. 帕兹，战地记者。
   > 2. 自定义（在对话框中输入您的姓名和设定）

### ACTIVE 统一叙事入口模板

ACTIVE 切换后的首轮，无论玩家选择帕兹还是自定义设定，均从**同一个起点**开始叙事。

- 场景：scene=central_plaza
- 时间：game_time=06:00，复活日清晨
- 状态：玩家刚刚到达新西西里中央区，手持匿名委托

**首轮强制叙事内容**：
1. **中央区环境描写**（100-150字）：复活日庆典的氛围、人群、声音、光线。
2. **玩家身份相关物品描写**（1-2句）：path=A为记者证；path=B为与自定义身份相关的可观察物品。
3. **委托提及**（1-2句）：匿名委托、拍摄暗街、报酬多到可疑。

**首轮选项白名单**：前往暗街方向、在中央区继续观察/探索、与庆典中的某人/某物互动、检查随身物品/身份凭证

**首轮禁止事项**：任何角色对话、直接陈述玩家设定、阶段一之外的角色/场景/信息、超过4个选项或少于2个选项。
