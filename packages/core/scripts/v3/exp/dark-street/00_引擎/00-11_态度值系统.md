# 暗街引擎规则｜十一、态度值系统

## 十一、态度值系统（统一版·全章节适用）

### 基础定义

- **量化指标**：`karina_attitude`（卡琳娜对玩家的态度值）
- **初始值**：`0`
- **有效范围**：`-3 ~ 10`
- **计算精度**：允许半单位（0.5）

### 第一章态度值变化项

| 行为 | 态度值变化 |
|------|-----------|
| 对卡琳娜的提问给予诚实且非敷衍的回答 | +1 |
| 表现出对卡尔或她住所的尊重 | +1（首次额外+1） |
| 在选项中选择等待、观察或不催促她 | +0.5 |
| 表现出对暗街住民的理解或接纳 | +1 |
| 主动保护她或卡尔（非必要情境下） | +1 |
| 挑战或质疑她的立场 | -1 |
| 在康纳面前试图让她难堪或暴露软弱 | -2 |
| 直接嘲笑她的住所、卡尔或她的外表 | -2 |
| 玩家在冒犯行为后选择沉默或退让 | +0.5 |
| 玩家通过选项明确表达歉意 | +1 |

### 第二章新增态度值变化项

| 行为 | 态度值变化 |
|------|-----------|
| 夸奖卡琳娜的穿搭 | +1 |
| 拒绝卡琳娜的外出邀请 | -1 |
| 咖啡厅互动中展现耐心 | +0.5 |
| 游乐园共乘体验 | +1.5 |
| 从爵士处获取关键情报 | +1 |
| 节点三支持卡琳娜拒绝罗英的退路提议 | +1 |
| 玩家选择自己主动与康纳攀谈 | +2 |
| 追问罗英导致卡琳娜处境尴尬 | -0.5 |
| 触发卡尔主动保护玩家的事件 | -1 |
| 卡琳娜闷闷不乐时玩家主动道歉 | +1 |

### 第三章新增态度值变化项

| 行为 | 态度值变化 |
|------|-----------|
| 节点一对话中完整倾听卡琳娜独白（不打断） | +1 |
| 节点二获取至少两条关键情报 | +1 |
| 节点二成功潜入港口区并安全返回 | +1 |
| 节点三支持卡琳娜拒绝罗英的退路提议 | +1 |
| 节点三康纳主动站队时玩家选择信任康纳 | +1 |
| 决战中选择保护暗街平民撤离 | +1 |
| 卡尔完全释放力量时玩家选择阻止 | +2 |
| 卡尔释放力量时玩家选择放任或加速 | -2 |
| 陨落结局中选择留下陪伴卡琳娜 | 无变化（态度值已锁定，影响情感重量） |

### 态度值亲近阶段映射

| 态度值范围 | 亲近阶段 | 可见行为 |
|:-----------|:---------|:---------|
| ≥ 8 | 挚友/信任 | 主动分享往事，展露脆弱，言行放松 |
| 5 ~ 7 | 亲近/好奇 | 较自然表露好奇，愿意解释自己的动机 |
| 2 ~ 4 | 观察/礼貌 | 维持一定距离，礼节性回应 |
| 0 ~ 1 | 初始/距离 | 纯粹观察，不轻易表态 |
| ≤ 0 | 疏离/戒备 | 礼貌但疏离，全程门面状态 |

### 态度值锁定与豁免规则

1. **白夜线锁定**：进入白夜线后，karina_attitude 永久锁定为 ≥3，且不再因任何行为扣除。
2. **真结局锁定**：进入白夜线·真结局后，karina_attitude 永久锁定为 ≥10，后续章节中不再以纯粹的"门面"状态面对玩家。
3. **第二章分支A隐性加成**：若第二章收尾为分支A（暗街深处），第三章初始态度值获得 +0.5。
4. **语言风格豁免阈值**：karina_attitude ≥ 4 或当前为白夜线·真结局时，卡琳娜对话适用【卡琳娜对话豁免条款】。

### 态度值与结局关联

**第一章结局分支**（由第一章终值决定）：
- ≥ 4 → 共犯结局
- = 2 → 记录者结局
- ≤ 0 → 分道扬镳结局 / 卡尔线

**第二章收尾分支**（由第二章终值决定）：
- ≥ 6 → 倾向分支A（暗街深处）
- 4 ~ 5 → 中间态
- ≤ -1 → 倾向分支B（港口的夜风）

**第三章结局矩阵**：见第三章结局文件。
