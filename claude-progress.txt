### Key Decisions & Lessons
- DeepSeek doesn't support response_format: json_schema (only json_object)
- @ai-sdk/openai-compatible v2 warns about responseFormat but still works
- @ai-sdk/openai v3 defaults to /responses endpoint (not /chat/completions)
- Switched from generateObject to generateText + manual JSON for reliability
- Made Goal5W1H.when optional since DeepSeek frequently omits it
- ScriptBundle atomic model: characters + chapters + goapActions bundled together for consistency
- idb library (v8.0.3) chosen over Dexie.js for minimal size (1.2KB) + TypeScript generics
- Dynamic GOAP actions are runtime-only (not persisted to IndexedDB) to prevent action library bloat
- AppView 3-state replaces binary isRunning for start/settings/game navigation
